A custom WordPress theme for EventTrashBox.

== Description ==

EventTrashBox is a custom WordPress theme built for publishing the site's posts, pages, archives, search results, and standard WordPress content.

== Installation ==

1. Place the `eventtrashbox` directory in `wp-content/themes`.
2. In WordPress admin, go to Appearance > Themes.
3. Activate EventTrashBox.
